moju

Physics you know, in the AI you train.

Dimensionless scaling, constitutive models, and equation residuals in one JAX library. For flow, heat transfer, and physics-informed ML.

pip install moju
Documentation GitHub PyPI

What's included

moju.piratio: Groups, Models, Laws, Operators (dimensionless scaling, physical models, conservation laws, differential operators). moju.monitor: ResidualEngine, build_loss, audit, visualize (residuals, physics loss, and monitoring). JAX-native, JIT-compiled, differentiable.

GroupsRe, Pr, Nu, Ma, …
Modelsμ(T), ρ(P,T), k(T)
Lawsmass, momentum, energy
Operators∇, ∇², ∇×, ∂/∂t
ResidualEngineresiduals, physics loss, audit, plots
